void ServicoUsuariosLeitor::consultarDefinicaoTermo ( )
virtual

Mais uma vez os termos da tabela precisam estar bem evidentes na mente de quem ira usar Essa funcao. Primeiro o programa realiza o questionamento de qual termo o usuario deseja consultar sua definicao. Apos isso uma verificacao ocorre a procura desse termo em duas tabelas juntas, processo conhecido no SQL como inner join, para assim podermos validar a existencia do termo na tabela Termo e na tabela Definicao. Tendo um resultado positivo nessa pesquisa, a tabela Definicao sera acionada, procurando e mostrando as informacoes do termo em questao.
